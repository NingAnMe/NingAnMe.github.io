本地建站

建站

安装完成后,就可以使用以下命令建站了

hexo 站点的目录结构如下

_config.yml

存放了网站的配置信息

themes 文件夹

存放列网站的主题文件,主题文件自己也有一个 _config.yml 文件,用来存放主题自己的配置信息

source 文件夹

_posts文件内存放了需要在主页展示的文章,_drafts存放了草稿文件,其他需要放到 github pages 的文件也放到 source 文件内,如 images 文件
