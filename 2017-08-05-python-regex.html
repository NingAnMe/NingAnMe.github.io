正则表达式 re 模块

常用方法

语法

字符 匹配
单个字符 —
. 匹配任意一个字符(除了\n)
[...] 匹配字符集中任何一个字符,如果匹配中括号,要加转义 \
\d/\D 匹配数字/非数字
\s/\S 匹配空白/非空白
\w/\W 匹配单词字符[a-zA-Z0-9]/非单词字符
多个字符 —
* 匹配前一个字符0次或者无限次
‘+’ 匹配前一个字符1次或者无限次
‘?’ 匹配前一个字符0次或者1次
{m}/{m, n} 匹配前一个字符m次或者匹配前一个字符m到n次
*? / +? / ?? 匹配模式变为非贪婪(尽可能少匹配字符)
边界匹配 —
^ 匹配字符串开头
‘$’ 匹配字符串结尾
\A/\Z 指定的字符串必须出现在开头或者结尾
分组匹配 —
` ` 匹配左右任意一个表达式
(ab) 括号中表达式最为一个分组
\<number> 引用编号为num的分组匹配到的字符串
(?P<name>) 分组起一个别名
(?P=name) 引用别名为name 的分组匹配字符串

re模块其他方法

search方法

在一个字符串中查找匹配,只找到第一个出现的位置

search(pattern, string, flags=0)

findall方法

找到匹配,返回所有匹配部分的列表

findall(pattern, string, flags=0)

sub方法

将字符串中匹配正则表达式的部分替换为其他值,然后返回字符串

sub(pattern, repl, string, count=0, flags=0)

repl 可以是一个函数

split方法

根据匹配分割字符串,返回分割字符串组成的列表

split(pattern, string, maxsplit=0, flags=0)
