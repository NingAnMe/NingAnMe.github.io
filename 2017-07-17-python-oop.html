Python 面向对象

  1. 面向对象的理论基础
  2. Python基本的面向对象特性

面向对象概念

类: 任何事物都可以归到一个具有某些共同特征的群体中,这个群体就称作一类

对象: 就是这个群体的其中之一

属性: 是对这些共同特征的详细描述

功能: 这个群体可以表现出来的行为

继承: 如果一个实例属于某一类,那么就会拥有这个类的共有属性和共有功能

多态: 每个实例都可以拥有自己区别与其它实例的属性和功能,对同一事物会有不同的处理方法

封装: 在别的类看来,只能知道其它类具有某些属性,和某些功能,并不知道它们的具体实现,这中特性就叫封装

父类: 被其它类继承了特性

子类: 继承了某个类的特性

多重继承: 一个类可以从两个和两个以上的类中继承

定义新式类

新式类比老式类具有更多的功能,当一个类不需要继承任何父类的时候,需要继承默认的object类,就会定义一个新式类,这个问题只存在与 Python2

关键词 class 用来定义一个类

Person 是定义类的名字

object 是继承的默认父类

__init__ 构造函数,是用来初始化实例用的方法

get_name() 是赋予类的一个方法

@property 是用来装饰一个方法用的,使方法可以像属性一样被使用

self 是类本身

name, age 是类的属性

self.name = name 用来赋予实例默认属性

self.__age = age 属性名称加上两个下划线就不能通过一般方法被类外面所访问

dir() 获取类的所有属性

type() 获取一个对象的类型

类的继承

子类会继承父类的属性和方法,也可以重新定义,覆盖掉父类的属性和方法

如果继承父类的属性和方法,需要使用 super() 函数

isinstance() 用来判断是否是某个类型

issubclass() 用来判断是否是子类

类的多态

做同一件事情拥有不同的方法和思想

要素: 继承,方法重写

Magic Method

方法名的前后有两个下划线

完整实例化一个对象的方法:

首先通过 new 方法创建对象名字

类与运算符

cmp(self, other)

eq(self, other)

lt(self, other)

gt(self, other)

add(self, other)

sub(self, other)

mul(self, other)

div(self, other)

or(self, other)

and(self, other)

如果需要定义类的对应功能,就要在类中实现对应的方法

类的展现

如果一个类具有 str repr unicode 三个方法,那么这个类就可以被转换为字符串

类的属性控制

setattr(self, name) 用来设置类的属性

self.__dict__[name] = value

getattr(self, name) 和 getattribute(self, name) 用来查询对象的属性

return super(Person, self).__getattribute__(name)

上面两类方法都容易引起无限递归,要注意正确的写法

delattr(self, name) 用来删除对象的属性
