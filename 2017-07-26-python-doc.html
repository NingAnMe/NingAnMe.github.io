Python 文件处理

发表于 2017-07-26

内容

  1. 文件的基本概念
  2. 掌握文件的基本操作
  3. 了解文件属性及linux 系统的文件管理机制
  4. 掌握os模块对文件和目录的处理

基本概念

文件: Python中文件是对象

linux文件: 一切设备都可以看成文件

例如: 磁盘文件,管道,网络Socket,外设等

Python文件基础操作

文件打开方式

f = open(name,) : 默认为只读方式

:

‘r’ : 只读方式,文件必须存在

‘w’ : 只写方式,文件不存在就创建,文件存在会清空内容

‘a’ : 追加方式,在文件结尾处追加内容

‘r+’ : 读写方式,不会清空内容

‘w+’ : 读写方式,会清空内容

‘a+’ : 追加和读写方式

二进制方式打开文件加一个 ‘b’ 如 ‘rb’ ‘wb’

关闭文件方法

f.close()

文件的读取方式

读取方式 :

表示读取多少字节

read() : 读取整个文件

readline() : 读取一行

readlines() : 读取一个 python_buffer 大小,返回每一行作为一个元素的列表

iter() : 使用迭代器真正读取整个文件

文件的写入方式

写入方式

write(str) : 将字符串写入文件

writelines(sequence_of_strings) : 写入一个字符串组成的序列(元组,列表等)到文件

写入文件内容有一个文件缓冲的过程

打开文件进行写入以后,调用关闭方式close()或者flush()以后,或者写入数据量大于写缓存的时候,缓冲的内容才真正写入文件

文件的关闭

将写缓存同步到磁盘

Linux 系统每个进程打开文件的个数是有限制的

打开文件以后, fileno 会加 1,如果大于限制,打开文件就会失败

所以必须养成良好的编码习惯,打开文件处理完毕以后一定要记得关闭

写入和读取问题

  1. 写入文件后,必须打开才能读取写入内容
  2. 读取文件后,无法重新再次读取读过的内容

文件指针操作:

seek(offset[, whence]) : 移动文件指针
offset : 偏移量,可以为负数
whence : 便宜相对位置

文件指针定位方式:

os.SEEK_SET : 相对文件起始位置

os.SEEK_CUR : 相对文件当前位置

os.SEEK_END : 相对文件结尾位置

f.tell() 返回当前文件的偏移

文件属性,OS模块使用

文件属性

f.fileno() : 文件描述符

f.mode() : 文件打开权限

f.encoding() : 文件编码格式

f.closed() : 文件是否关闭

标准文件

sys.stdin 标准输入

sys.stdout 标准输出

sys.stderr 标准错误

sys.argv 通过该属性可以得到命令行参数,是字符串组成的列表

文件的编码方式

python2 默认创建的文件是ACILL格式,需要转换为 utf-8 格式才能输入中文

a = unicode.encode(u’你好’, ‘utf-8’)

codecs 模块可以指定打开文件的编码方式

Linux 文件系统简介

Python 解释器得到一个对象以后,首先通过 SystemApI 获得它的文件描述符(文件节点),然后通过 SystemCall 调用虚拟文件系统,虚拟文件系统会调用对应设备驱动,再对硬件进行操作

OS 模块对文件进行操作

打开文件

os.open(filename, flag, )

flag:
os.O_CREAT: 创建文件
os.O_RDONL: 只读方式打开
os.O_WRONLY: 只写方式打开
os.O_RDWR: 读写方式打开

读取文件

os.read(fd, buffersize)

写入文件

os.write(fd, string)

文件指针操作

os.lseek(fd, pos, how)

关闭文件

os.close(fd)

os模块常用方法名 作用
os.access(path, mode) 判断该文件权限:F_OK 存在,R_OK,W_OK,X_OK,读,写,执行
os.lisdir(path) 返回当前目录下所有文件组成的列表
os.remove(path) 删除文件
os.rename(old,new) 修改文件或者目录名
os.mkdir(path,) 创建目录
os.makedirs(path,) 创建多级目录
os.remobedirs(path) 删除多级目录
remdir(path) 删除目录(目录必须空目录)
os.path模块常用方法名 作用
exists(path) 当前路径是否存在
isdir(path) 是否是一个目录
isfile(path) 是否是一个文件
getsize(filename) 返回文件大小
dirname(path) 返回路径的目录
basename(path) 返回路径的文件名
